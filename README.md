# ☕ روز كافيه — نظام الطلب عبر QR

نظام طلب ذكي للمقاهي والمطاعم عبر مسح رمز QR من الطاولة.

## 🚀 المميزات

### للزبون
- 📱 مسح QR → يفتح قائمة الطاولة
- 🍽️ تصفح القائمة بالتصنيفات مع بحث
- 🛒 سلة مشتريات مع خيارات وملاحظات
- ✅ تأكيد الطلب وعرض رقم الطلب

### للإدارة
- 🔐 تسجيل دخول آمن
- 📋 لوحة طلبات لحظية (SSE) مع تنبيه صوتي
- 🔄 تغيير حالة الطلب: جديد → تحضير → جاهز → مقدم
- 🍽️ إدارة القائمة (تصنيفات + أصناف + توفر)
- 📱 مولد QR للطاولات مع طباعة
- 📊 إدارة الطاولات (إضافة/تعطيل/حذف)

## 🛠 التقنيات

| التقنية | الاستخدام |
|---------|----------|
| Next.js 14 | Frontend + API |
| Prisma | ORM |
| PostgreSQL | قاعدة البيانات |
| Tailwind CSS | التصميم |
| SSE | تحديثات لحظية |
| Docker | بيئة التطوير |

## ⚡ التشغيل

### المتطلبات
- Node.js 18+
- Docker & Docker Compose

### الخطوات

```bash
# 1. استنساخ المشروع
cd "rose cafe"

# 2. نسخ ملف البيئة
cp .env.example .env

# 3. تشغيل PostgreSQL
docker-compose up -d

# 4. تثبيت التبعيات
npm install

# 5. تطبيق الهجرات
npx prisma migrate dev

# 6. تغذية البيانات
npx prisma db seed

# 7. تشغيل الخادم
npm run dev
```

### الروابط
| الصفحة | الرابط |
|--------|--------|
| الصفحة الرئيسية | http://localhost:3000 |
| قائمة طاولة 1 | http://localhost:3000/t/1 |
| تسجيل دخول الأدمن | http://localhost:3000/admin/login |
| لوحة الطلبات | http://localhost:3000/admin/orders |
| إدارة القائمة | http://localhost:3000/admin/menu |
| QR الطاولات | http://localhost:3000/admin/tables |

### بيانات الأدمن الافتراضية
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`
